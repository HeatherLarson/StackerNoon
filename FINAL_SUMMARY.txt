                          ğŸš€ GETTING STARTED

For Readers:
  1. Open StackerNoon
  2. Browse featured and recent articles
  3. Filter by category (#bitcoin, #ethereum, #nostr, etc.)
  4. Click any article to read full content
  5. See author Nostr profile

For Publishers:
  1. Create Kind 23 event in any Nostr client
  2. Add title, summary, content (markdown)
  3. Tag with categories (#bitcoin, #ethereum, etc.)
  4. Publish to Nostr relays
  5. Article appears on StackerNoon instantly!

For Developers:
  1. Clone the repository
  2. npm install to install dependencies
  3. npm run dev to start development server
  4. Edit components in src/components/blog/
  5. Edit hooks in src/hooks/
  6. Customize colors in src/index.css
